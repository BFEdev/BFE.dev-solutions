# Task: #37. push unshift

## Tags: 

JavaScript

## Difficulty:

Easy

## Question:

What does the code snippet output by console.log?

## Explanation:

```javascript
const arr = [1, 2];
arr.push(3, 4);
arr.unshift(5, 6);
console.log(arr); // [5,6,1,2,3,4] - just read about push() and unshift()
```

## Answer:

```javascript
[5,6,1,2,3,4]
```
